## ✨ About

**Future You 2030** is a one-page interactive web experience built for the **Weboreel AI Hackathon**. It allows users to explore their potential future through engaging storytelling and simple interaction flows.

Users can choose different life paths — **Student**, **Startup Founder**, or **Content Creator** — and instantly visualize their journey to 2030 with a personalized timeline, stats, and narrative.

---

## 🎯 Key Features

| Feature | Description |
|---|---|
| 🎓 Interactive Path Selection | Choose between 3 unique life paths |
| 📊 Dynamic Stats & Timeline | Personalized year-by-year journey to 2030 |
| ✨ Particle Background | Floating connected particles for a futuristic feel |
| 🎵 Ambient Music | Immersive background audio with toggle control |
| 📱 Fully Responsive | Works seamlessly on mobile, tablet, and desktop |
| ⌨️ Keyboard Accessible | Full keyboard navigation and ARIA labels |

| ⚡ Zero Dependencies | Pure HTML, CSS, and JavaScript — no frameworks |

## 🛠 Getting Started

### Prerequisites

Just a modern web browser. That's it. No build tools, no npm, no frameworks.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Tanya-garg10/creative01-future-you-2030.git

# Navigate to the project
cd creative01-future-you-2030

# Open in browser (pick one)
start index.html          # Windows
open index.html           # macOS
xdg-open index.html       # Linux

# Or use a local server
python -m http.server 8080
# Then visit http://localhost:8080
```

## 🧠 How It Works

1. **Hero Section** — User lands on a futuristic welcome screen
2. **Choose Your Path** — Pick between Student, Startup Founder, or Content Creator
3. **Future Revealed** — Dynamic content shows personalized 2030 story + stats
4. **Timeline** — Year-by-year journey from 2025 to 2030
5. **Call to Action** — Explore another path or restart

All content is generated dynamically via JavaScript — no page reloads, no external APIs.
